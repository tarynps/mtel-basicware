# V2 Executive Deck Structure

`index-v2.html` is a parallel executive deck. It uses the current project as a component library and visual reference while keeping `index.html` untouched.

## Commercial Proposition

**Mtel × Basicware operates the AI growth system for enterprises, powered by BytePlus.**

## Core Value Loop

Generate content → Personalize → Measure → Learn → Convert

## Consolidation from Current Deck

- Current Slides 02–05 are consolidated into V2 Slides 03–05.
- Current Slides 07–09 are reframed into V2 Slides 04, 07, and 08.
- Current industry, pricing, and broad capability-list slides are removed from the executive flow.
- Current outcomes and case-study ideas are consolidated into V2 Slide 09 with cautious, measurable framing.
- Current partnership and engagement slides are consolidated into V2 Slides 10–11.

## Design Direction

- Premium white, yellow, and navy palette.
- Larger whitespace and fewer words per slide.
- Varied layouts: split views, role cards, loops, workflow maps, interactive preview, and decision close.
- Global typography tokens from `globals.css`.
- Independent V2 navigation, overview, presenter notes, and interactive preview.
